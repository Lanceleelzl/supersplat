# SuperSplat 基础操作指南 (Official Guide)

本文档介绍 SuperSplat 编辑器的基础操作与原生功能。

## 1. 视图控制 (Camera Controls)

| 操作 | 鼠标行为 | 说明 |
| :--- | :--- | :--- |
| **旋转 (Orbit)** | `左键拖拽` | 围绕当前焦点旋转相机 |
| **平移 (Pan)** | `右键拖拽` 或 `中键拖拽` | 平移相机视角 |
| **缩放 (Zoom)** | `滚轮滚动` | 拉近或推远视角 |
| **聚焦 (Focus)** | `双击对象` | 将相机聚焦到选中的 Splat 或对象上 |

## 2. 工具栏 (Toolbar)

位于编辑器顶部的工具栏提供了最常用的操作模式：

### 2.1 变换工具 (Transform Tools)
- **移动 (Move)**: 快捷键 `1`。显示三轴坐标轴，拖拽轴向移动选中对象。
- **旋转 (Rotate)**: 快捷键 `2`。显示旋转球，拖拽圆环旋转选中对象。
- **缩放 (Scale)**: 快捷键 `3`。显示缩放轴，拖拽轴向缩放选中对象。

### 2.2 选择工具 (Selection Tools)
- **矩形选择 (Rect)**: 快捷键 `R`。拖拽框选屏幕范围内的 Splat 点。
- **笔刷选择 (Brush)**: 快捷键 `B`。像画笔一样涂抹选择区域。
- **多边形选择 (Polygon)**: 点击绘制多边形区域进行选择。
- **套索选择 (Lasso)**: 自由绘制闭合区域进行选择。
- **球形选择 (Sphere)**: 在 3D 空间中通过球体范围进行选择。

## 3. 场景管理 (Scene Management)

- **加载模型**: 直接将 `.ply` 或 `.splat` 文件拖入窗口即可加载。
- **场景面板**: 右侧面板列出了当前场景中的所有对象，点击眼睛图标 `👁️` 可控制显示/隐藏。
- **属性面板**: 选中对象后，右侧会显示其详细属性（位置、旋转、缩放等）。

## 4. 快捷键一览 (Shortcuts)

- `Ctrl + Z`: 撤销
- `Ctrl + Y` / `Ctrl + Shift + Z`: 重做
- `Space`: 切换选择/移动模式
- `Delete` / `Backspace`: 删除选中对象
- `Esc`: 取消选择
- `H`: 隐藏选中对象
- `Shift + H`: 显示所有隐藏对象
